# ExpressJS - CRUD API (example-1)

A goal API powered by ExpressJS, MongoDB, and Mongoose object data modeling (ODM).

<br />

## How to Run

1. Install dependencies

```
yarn
```

<br />

2. Set environment variables in `.env`

```
cp .env.example .env
```

<br />

3. Run the app

```
yarn dev
```
